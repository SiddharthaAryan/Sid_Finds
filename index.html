API learning roadmap

This is how we will convert the MVP into a live connected system.

1

CSV MVP

First we prove the scoring logic with exports. No keys, no secrets, no backend needed.

2

Google Search Console API

Next we connect query, page, clicks, impressions, CTR and average position through OAuth.

3

GA4 Data API

Then we pull users, sessions and conversions by landing page to understand real business value.

4

RankNow integration

If RankNow provides API access, we connect it. If not, scheduled CSV upload remains practical.

5

Trends layer

Google Trends can be added carefully, but we keep it separate because official API support is limited.

Built for digital healthcare marketing decisions. Keep private API keys out of frontend code.
